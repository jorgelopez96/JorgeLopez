## 🚀 Proyectos destacados

### 💜 [CuidaTuWallet](https://cuida-tu-wallet.vercel.app)
> App de finanzas personales · React + Firebase + Tailwind

Seguimiento de gastos, gestión de tarjetas de crédito, dashboard con gráficos, modo oscuro/claro, tooltips de onboarding y expiración de sesión. Inspirado en el UX de apps como Mercado Pago y Brubank.

`React` `Vite` `Firebase` `Tailwind CSS` `Vercel`
